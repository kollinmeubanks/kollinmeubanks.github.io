# Georgia Stingerz SquadLocker Store

This is a GitHub Pages site that hosts a branded Georgia Stingerz landing page and loads the SquadLocker store widget.

## Files

- `index.html` - main website file
- `logo.png` - optional Stingerz logo file

## GitHub Pages Setup

1. Upload `index.html` and your `logo.png` file to the repository.
2. Go to Settings > Pages.
3. Choose Deploy from branch.
4. Select `main` and `/root`.
5. Save.

Your site will publish at:

`https://YOUR-USERNAME.github.io/YOUR-REPOSITORY-NAME/`
